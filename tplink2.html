Cara Setting Wifi pada Router Tp-Link

Silahkan perhatikan langkah-langkah berikut.

1. Pastikan Hp anda terhubung ke jaringan wifi

2. Buka Alamat 192.168.0.1 pada Browser Google Chrome atau Klik disini

3. Masukkan Password login apabila ada.
* Default Password = admin888

4. Pilih 3 Garis dibagian pojok kiri atas

5. Pilih No.3 Wireless Setting>

6. Network Name - SSID (untuk mengganti nama wi-fi)
7. Password (untuk mengganti sandi wi-fi)

8. Lalu Klik Save di pojok kanan atas

Selesai.
apabila ada yang perlu ditanyakan Silahkan Klik tombol Chat diatas.
